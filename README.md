# CVS Exercise

## Install

```sh
clone or download repository
```

```sh
npm install 
```

```sh
npm run prebuild:dev 
```

```sh
npm run build:dev
```

```sh
npm start 
```

## Temporary Web Server
From working directory type:

$ npx http-server --cors
